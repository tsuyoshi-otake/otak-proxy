# プロダクト概要

otak-proxyは、VSCodeでのプロキシ設定をワンクリックで管理するための拡張機能です。Git、VSCode、npm、および統合ターミナルのプロキシ設定を一元管理し、企業環境やネットワーク制限のある環境での開発効率を向上させます。

## コア機能

1. **3モード切替**: Off / Manual / Auto の切り替えをステータスバーからワンクリックで実行
2. **システムプロキシ自動検出**: Auto モードでシステム/ブラウザのプロキシ設定をリアルタイム同期
3. **接続検証**: プロキシ有効化前に接続性をテスト
4. **統合管理**: Git、VSCode設定、npm、統合ターミナル環境変数を一括設定
5. **フォールバック機能**: システムプロキシが利用不可の場合、手動設定にフォールバック

## 対象ユースケース

- **企業環境での開発**: 社内プロキシを経由した外部アクセスが必要な環境
- **ネットワーク切替**: オフィス/リモートでのプロキシ設定切替が頻繁な場合
- **初期設定の簡素化**: プロキシ設定の手動管理の煩雑さを解消

## 提供価値

- **ゼロコンフィグ**: システムプロキシの自動検出により設定不要
- **一括管理**: 複数ツールのプロキシ設定を一元化
- **リアルタイム同期**: システムプロキシ変更の自動追従
- **多言語対応**: 日本語/英語 UI
